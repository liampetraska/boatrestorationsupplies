Folder Structure

Files:

index.html: for choose type of template like image, slider or video background.

image-background.html: html file for image background template

slider-background.html: html file for slider background template

video-background.html: html file for youtube video background template.

email.php: a php file which has three options for save your subscribers email.

 

Folders:

css: all css files

Images: all Images

js: all js libraries

vegas:javascript and css file for vegas slider

Count Down

to change countDown go to folder js and file script.js

// JAVASCRIPT COUNT DOWN ============================================================
	$('.countdown').downCount({
		date: '06/10/2014 12:00:00',
		offset: +10
	});

change Date and Hour

Image Background

to change image background go to folder css and file style.css

body {
	background-image: url(../images/image_01.jpg);
}

Slider Background

to change slider background go to folder js and file slider.js

$.vegas('slideshow', {
		delay: 3000,
		backgrounds: [
			{ src:'./images/image_01.jpg', fade:1000 },
			{ src:'./images/image_02.jpg', fade:1000 },
			{ src:'./images/image_05.jpg', fade:1000 }
		]
	})('overlay', {
		src: './vegas/overlays/06.png'
	});

Video Background

PHP Subscription Form (PHP5)

If you want to change the filename of PHP file, don't forget to change it in script.js to.

	url:'xxxx.php'

You have three options for save your subscribers email in file email.php:

  • Save to a txt file
  • Send to your email account
  • Save to database (MySQL)

By default all above options are disabled, to enable one or more options, just set to "TRUE".

// Line 23
$saveTxt   = FALSE;
// Line 26
$sendEmail = FALSE;
// Line 29
$saveMySQL = FALSE;
For saveTxt:

*Note: The file will created for you if not exist!

// Line 55
$file      = "xxx.txt";
$delimiter = "\r\n";
For send Email: 
// Line 73
$to      = "xxx@xxx.com";
$subject = "From Subscription Form";
For saveMySQL:

*Note: BEFORE using this option, you have to create only the database in MySQL, using phpMyAdmin as example, the table will created for you if not exist!

// Line 96
$server   = 'localhost';
$username = 'xxx';
$password = 'xxx';
$database = 'xxx';  // You have to create it first!
$table    = 'xxx';
Please DON'T change anything else, unless you know!
